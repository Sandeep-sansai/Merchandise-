Component Styling (home.component.css) /* Navbar */ .navbar { display: flex; justify-content: space-between; } .spacer { flex: 1; } /* Slideshow */ .slideshow-container { overflow: hidden; position: relative; } .slideshow-container img { width: 100%; height: 300px; object-fit: cover; transition: opacity 1s ease-in-out; } /* Hero Section */ .hero { text-align: center; margin: 20px auto; padding: 40px 20px; background-color: #f8f9fa; } .hero h1 { font-size: 2.5rem; margin-bottom: 15px; } .hero p { font-size: 1.2rem; margin-bottom: 20px; } .hero button { font-size: 1rem; } /* Footer */ .footer { background-color: #343a40; color: white; padding: 20px; } .footer h3 { color: #ffd700; } .footer button { text-transform: capitalize; } 3. Component Script (home.component.ts) import { Component, OnInit } from '@angular/core'; @Component({ selector: 'app-home', templateUrl: './home.component.html', styleUrls: ['./home.component.css'], }) export class HomeComponent implements OnInit { currentSlide: number = 0; totalSlides: number = 3; constructor() {} ngOnInit(): void { this.startSlideshow(); } startSlideshow(): void { setInterval(() => { this.currentSlide = (this.currentSlide + 1) % this.totalSlides; }, 5000); // Change slide every 5 seconds } } 4. Angular Material Modules Make sure to import the necessary Angular Material modules into your application. Add them to your AppModule: import { NgModule } from '@angular/core'; import { MatToolbarModule } from '@angular/material/toolbar'; import { MatButtonModule } from '@angular/material/button'; import { MatGridListModule } from '@angular/material/grid-list'; @NgModule({ declarations: [], imports: [ MatToolbarModule, MatButtonModule, MatGridListModule, ], exports: [ MatToolbarModule, MatButtonModule, MatGridListModule, ], }) export class MaterialModule {} Include the MaterialModule in your AppModule: import { MaterialModule } from './material.module'; @NgModule({ imports: [ MaterialModule, // Other imports... ], }) export class AppModule {} Explanation 1. Navbar: • mat-toolbar provides the navbar with a primary color theme. • Buttons within the navbar are created using mat-button and styled to align horizontally. 2. Slideshow: • Uses mat-grid-list to display a slideshow. • The currentSlide is updated every 5 seconds using setInterval, which changes the image dynamically. 3. Hero Section: • Contains a header, description, and a mat-raised-button for the “Book Tickets” button. 4. Footer: • Built with a two-column layout using mat-grid-list, providing sections for Quick Links and Contact Info. 5. Slideshow Logic: • The slideshow dynamically changes the displayed image by incrementing the currentSlide index. Running the Application 1. Ensure the Angular Material library is installed: ng add @angular/material 2. Add images to the assets/images folder: • slide1.jpg • slide2.jpg • slide3.jpg 3. Serve the application: ng serve You’ll have a responsive home page with Angular Material components, including a navbar, slideshow, hero button, and footer!
